
# 🖥️ Sistema Operacional com CRUD em `.txt`

Sistema Operacional que simula um sistema básico com funcionalidades de CRUD (Create, Read, Update, Delete) utilizando arquivos `.txt` como base de dados.

---

## 📋 Funcionalidades

- ✅ **Criar** registros e armazená-los em um arquivo `.txt`
- 🔍 **Ler** dados de forma organizada
- ✏️ **Atualizar** informações existentes
- ❌ **Deletar** registros do sistema
- 💾 Persistência de dados em um arquivo `.txt`

---

## 🛠️ Tecnologias Utilizadas

- Linguagem: `C`
- Armazenamento: Arquivos `.txt`
- Interface: Terminal

---

## 🚀 Como Executar

Clone o repositório:
`git clone https://github.com/MathMoura18/txt-OS.git`
`cd txt-OS`

Compile os códigos no Terminal:
`make`

Abra em Terminal separados do servidor e cliente:
`./servidor`
`./cliente`

Escreva as instruções disponíveis no Terminal do Cliente e veja os retornos no Terminal do Servidor.

Instruções:

- SELECT:  ` SELECT id=?`
- INSERT:  ` INSERT id=? nome=?`
- UPDATE:  ` UPDATE id=? nome=?`
- DELETE:  ` DELETE id=?`
